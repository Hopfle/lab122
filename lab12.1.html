Команди Git

Створити репозиторій з нуля

$ git init [project name]

Завантажити з існуючого репозиторію

$ git clone my_url

Перейти на гілку

$ git checkout my_branch

Створити нову гілку

$ git branch new_branch

Видалити гілку

$ git branch -d my_branch

Переглянути список змін

$ git status

Зробити коміт

$ git commit -m "commit message"

Отримати останні зміни з origin

$ git pull

Відправити локальні зміни до origin

$ git push
